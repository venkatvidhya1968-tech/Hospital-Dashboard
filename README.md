Hospital Dashboard project

This project aims to design a data-driven hospital dashboard to monitor:
- Patient admissions and discharges
- Bed occupancy rates
- Department-wise resource utilization
- Staff availability
- Patient outcomes

Tools & Technologies Used:
- Power BI (Dashboard & Visualization)
- PostgreSQL / MySQL (Sample Dataset Design)
- Excel / CSV (Data Source)

This dashboard helps hospital administrators make informed operational decisions.
